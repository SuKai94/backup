#####起初，我使用Wordpress for SAE搭建博客，转战到这里重新开始搭建博客的理由有：

- **Wordpress模板太少，否则就是要钱的**
- **把博客搭建在SAE上，每天消耗我大量云豆**
- **Wordpress对Markdown的支持不是很完美**
- **喜欢使用目前这种简洁的风格**
- **只需要把文章git到github，即活跃了博客，更活跃了github**
